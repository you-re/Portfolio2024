Sonica Animation

About

This project was created as a submission for Sonica, an annual music festival in Slovenia. It explores Houdini’s latest soft body and rigid simulation tools, newly integrated into Vellum, pushing the boundaries of procedural animation and dynamic simulations.

Gallery

Tools Used

- Houdini: Simulation, Modeling, Unwrapping, Lighting
- Octane: Lighting, Rendering
- Blender: Modeling, Unwrapping
- Substance Painter: Texturing
- DaVinci Fusion: Compositing, Color Grading
